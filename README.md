# Capital Cities of India

This project was created as part of my **Day 3 internship assignment**, where I was introduced to Angular development.

The project started with creating a basic Angular application and understanding its project structure. As part of the learning process, I explored Angular components, routing, Bootstrap integration, and custom styling while building a simple application based on the theme **"Capital Cities of India."**

The application includes multiple pages such as Home, About, and Contact, along with Bootstrap components including a navigation bar, carousel, accordion, and contact form.

## Technologies Used

- Angular
- TypeScript
- HTML
- CSS
- Bootstrap 5

## Features

- Home, About, and Contact pages
- Angular Components
- Angular Routing
- Bootstrap Navbar
- Carousel
- Accordion
- Contact Form

## Running the Project

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
ng serve
```

3. Open your browser and visit:

```
http://localhost:4200
```
